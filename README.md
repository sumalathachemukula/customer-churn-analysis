# Customer Churn Analysis

## 📌 Project Overview
This project aims to identify key factors that contribute to customer attrition (churn). By analyzing customer demographics, account information, and usage patterns, this Power BI dashboard helps businesses understand why customers leave and how to improve retention rates.

## 📁 Repository Contents
* **churn.pbix**: The interactive Power BI dashboard containing the data visualizations and DAX measures.
* **Customer Churn Analysis.csv**: The dataset used for the analysis.
* **Dashboard Preview**: A screenshot of the finalized report.

## 📊 Key Analysis Areas
* **Churn Rate:** Percentage of customers leaving the service.
* **Demographic Insights:** Analysis by age group, gender, and location.
* **Contract & Payment types:** Evaluating how different billing methods impact loyalty.
* **Service Usage:** Identifying which specific services are linked to higher churn.
